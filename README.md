# SnapCd.Common

Common DTOs, enums, and shared types for clients such as [SnapCD.Runner](https://github.com/schrieksoft/snapcd-runner) to interact with [snapocd.io](https://snapcd.io).

## Installation

```bash
dotnet add package SnapCd.Common
```

## Contents

This package contains:

- **DTOs** - Data transfer objects for API communication
- **Enums** - Shared enumeration types
- **Constants** - Endpoint and discriminator constants
- **Runner Requests** - Request/response types bidirectional Runner-to-Server communication (via SignalR)
